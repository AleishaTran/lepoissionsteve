Natanya, Aleisha, Katie

Report Code Demo

LePoisson PacMan project uses reinforcement learning techniques to enable an AI agent to perceive and navigate the game environment intelligently, learning optimal strategies through trial and error.

Abstract

In this project, we explore the application of reinforcement learning to train an AI agent to play the classic Pac-Man game. Our system learns optimal behaviours through interaction with the game environment, using neural networks to make decisions based on state observations. This approach enables the agent to improve its performance over time, demonstrating intelligent navigation and strategy in a dynamic, adversarial setting.
